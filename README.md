# Usage
## Backend
Make sure you have `API.key` in `backend/`.
```sh
cd backend
flask run --debug
```

## Frontend
Make sure you set `FLASK_BACKEND_URL` to the URL of the backend in `.env.local`.
```sh
npm install
npm run dev
```
